A tour of fun CSS

CSS can sometimes be a bit intimidating to some people, even now with CSS grid. However, CSS can be quite nice looking, so here are several examples of pretty looking color effects one can create in pure CSS.

All these colors have been created using the native linear or radial gradient feature that CSS has an option. This allows one to create neat color mixes into eachother, that can be useful from nice backgrounds, to making colorful text, but thats not a common supported option at the moment.
linear-gradient(to right, #C54B8C, #ff6600)
linear-gradient(to right, #ae3ab4, #45ddfc)
linear-gradient(to right, #3ab2b4, #e0fc45)
radial-gradient(farthest-corner at 10px 10px, #e1d46f 40%, #c52a74)
radial-gradient(#70b2df, #57dd7a)
radial-gradient( farthest-side circle at 10px 10px, #B284BE 40%, #88dc94)
